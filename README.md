# Expo

Es un framework utilizado para la construccion de aplicaciones moviles nativas, utilizando el framework `React JS`.

- Compila en codigo nativo.
- Opera a traves de un puente que conecta la logica de JavaScript con componentes nativos y API's especificas de los componentes.

## Creacion del proyecto

```bash
npx create-expo-app@latest <app_name>
```

1. Instalacion de dependencias

   ```bash
   npm install
   ```

2. Iniciar la aplicacion

   ```bash
    npx expo start
   ```

## Limpiar los directorios y archivos del proyecto

```bash
npm run reset-project
```
